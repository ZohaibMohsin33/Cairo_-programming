# Guide

## Overview
This project is a small Cairo exercise focused on loops.
It prints the numbers 1 through 10 and computes their sum.

## How it works
The `main` function in `src/loops.cairo` sets a mutable counter and sum.
It uses a `while` loop to print each number, add it to the running total,
and then increment the counter.
After the loop finishes, it prints the final sum.

## How to run
1. Open a terminal.
2. cd into the `Ebad_Junaid_Loops` folder.
3. Run: `scarb execute`

## Expected output
Numbers are printed on separate lines from 1 to 10, then:
Sum = 55
